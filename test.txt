1а Дано: интернет магазин на "1с-битрикс: Бизнес", каталог, корзина и оформление заказа реализованы нативными компонентами (catalog, sale.basket.basker, sale.order.full). 
Задача:  реализовать акцию "Каждый N товар за X рублей". У акции должны быть настройки, где выставляется период активности акции, параметры N и X

Если заказ производится в период активности акции, то:

1. Сортируем товары по увеличению цены, можно создать массив товаров $ar1, где ключ = id товара, а значение - его цена, и второй массив $ar2 с ключом также id, значение - количество экземпляров товара

2. Делим целочисленно количество товаров на N:  $C = floor(Count/N);

3. Перебираем массив $ar1 (с помощью foreach). При каждом проходе из количества экземпляров товара вычитаем $C:
 - если получившееся значение >= 0 то $C экземплярам данного товара назначаем цену X руб, прерываем перебор массива (breack)
 - если значение < 0, то $C приравниваем получившемуся значению (его абсолютному значению), т.е. тому сколь экземпляров осталось, всем экземплярам данного товара назначаем цену X руб. -> Переходим к следующему элементу
